Privacy Policy

IEEE PRISMTECH Privacy Policy

Participant, sponsor, and enquiry information is collected only for event registration, operations, communication, certificates, safety, and reporting.

Data We Collect

Names, roll numbers, college IDs, contact details, institution information, team details, uploaded ID documents, project links, and enquiry messages.

How Data Is Used

Data supports duplicate registration prevention, eligibility checks, team dashboard access, announcements, attendance tracking, certificates, sponsor enquiries, and event safety.

Retention And Security

Production deployment should store data in a secured database with role-based access, audit logs, file access controls, CAPTCHA protection, and encrypted transport.
